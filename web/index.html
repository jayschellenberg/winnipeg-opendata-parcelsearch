How to search River Lots, Outer Two Mile, and Section-Township-Range parcels

Unusual legal descriptions go into the same four fields, just with different values:

  • River Lots: put RL in the Block field and the parish code in the Plan field (e.g. JA for St. James, CH for St. Charles — or another parish). The Lot field takes the river-lot number.
  • Outer Two Mile lots: put OT in the Block field and the parish code in the Plan field.
  • Section-Township-Range: for a description like NW 10-11-4E (NW ¼ of Section 10, Township 11, Range 4E), put the section in Lot (10), the township in Block (11), the range in Plan (4E), and optionally the quarter (NW) in Description.

Partial matches work in every field.
